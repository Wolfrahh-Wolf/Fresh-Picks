Algorithmic Core (DSA)

Singly Linked List (SLL)
Time: O(N) | Space: O(N)

Serves as the foundational memory architecture. Dynamically deserializes database records directly from flat binary .dat files at runtime.

Doubly Linked List (DLL)
Time: O(1) Deletions

Drives the real-time shopping cart. Bi-directional node traversal enables instant mid-list item removals without array-shifting overhead.

Min-Heap (Priority Queue)
Time: O(log N)

Powers the Admin Dispatch Engine. Sorts queued orders by delivery urgency (e.g., Morning slots bubble to the root) for immediate extraction.

Circular Linked List (CLL)
Infinite Loop Routing

Facilitates the logistics engine. Implements a wrap-around round-robin algorithm to guarantee mathematically equitable order assignment.

Pointer Table Algorithm
Time: O(1) Access

Custom lookup table. Parses alphanumeric IDs (U1042 → 42) into array indices to instantly map to specific memory addresses, bypassing SLL searches.
